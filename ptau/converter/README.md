# ptau.converter



This package stores some tools for conversion between different formats. 



## Functions

ptau.converter.**midicsv**(*midifile, csvfile*)

​	converts midi file into csv file.



ptau.converter.**csvmidi**(*csvfile*, *midifile*)

​	Not Implemented Yet. converts csv file into midi file.
